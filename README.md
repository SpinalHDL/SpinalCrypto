# SpinalCrytpo


This repository contains cryptographic Spinal IPs. 

## IP available

### Symmetric

- DESCore_Std
- TripleDESCore_Std


### Hash

- HMACCore_Std (tested with MD5)
- MD5Core_Std

Documentation is available on the [wiki](https://github.com/SpinalHDL/SpinalCrypto/wiki) page.
